TellerFlow AI: Work Faster, Stay Accurate

This page gives you a practical structure to build an AI assistant for teller work: customer intake, transaction checklists, compliance reminders, and end-of-day handover notes.

What this AI should do

1) Pre-Transaction Guidance

  • Ask for transaction type and amount.
  • Show required documents before processing.
  • Flag limits and approval thresholds.

2) Real-Time Checklist

  • Generate a step-by-step teller checklist.
  • Include verification, counting, and confirmation steps.
  • Reduce missed steps during rush hours.

3) Compliance Guardrails

  • Prompt for KYC/AML checks based on transaction risk.
  • Remind when supervisor override is required.
  • Generate a short audit-ready note.

4) Shift Reporting

  • Summarize pending issues before handover.
  • List unusual transactions and actions taken.
  • Prepare a clean end-of-day report.

Starter Prompt Template

"You are a bank teller assistant. For each request, provide: (1) required checks, (2) step-by-step process, (3) compliance alerts, (4) customer-friendly response text, and (5) end note for audit logs."
