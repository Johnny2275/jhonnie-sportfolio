# Portfolio Starter Web App

A simple, responsive single-page portfolio template built with plain HTML, CSS, and JavaScript.

## Run locally

Open `index.html` directly in your browser, or serve with a local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Customize

- Replace `Your Name` in `index.html`.
- Update project entries in `script.js`.
- Replace the Formspree action URL in the contact form with your own endpoint.
- Update social links in the footer.
